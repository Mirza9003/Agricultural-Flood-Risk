## Overview

This repository presents a machine-learning-driven agricultural flood risk assessment framework for the state of Iowa. The workflow integrates flood susceptibility modelling, crop exposure, and socioeconomic vulnerability to support spatial prioritization of flood mitigation. The framework combines multi-source geospatial datasets, explainable GeoAI, and an interactive Google Earth Engine application for visualization and decision support.

---

## Key Capabilities

- Machine learning-based flood susceptibility mapping at 30 m resolution
- Evaluation of gradient boosting models including XGBoost, LightGBM, and Histogram GBM
- Integration of hazard, crop exposure, and socioeconomic vulnerability
- Explainable AI interpretation using SHAP
- Statewide analysis for Iowa with county-level relevance
- Interactive cloud deployment using Google Earth Engine
- Open-access archived dataset via Zenodo
